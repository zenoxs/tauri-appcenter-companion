# Tauri AppCenter Companion

A companion app written with **React** aand **Tauri** to help manage your applications on appcenter.

![Main App Light](./screenshots/main_light.png)
<!-- ![Main App Dark](./screenshots/main_dark.png) -->

## Install
```shell
yarn install
```


## Start app in dev

```shell
yarn run tauri dev
```
